### 30-Second Verbal Script

“Let me restate the problem to make sure I’m aligned.
We’re given ___ and we need to return ___.

Before coding, I want to clarify a few things.
I’m assuming the input fits in memory, can include negative numbers, and that correctness plus efficiency matter. Does that sound right?

There are a couple ways to approach this.
The straightforward way is ___, which is easy to reason about but costs ___ in time/space.
A more efficient approach is ___, which improves ___ but is more complex.

Given the constraints and tradeoffs, I’d go with ___, and I’ll make sure to handle edge cases like ___.”

### Practice tips

* Always **fill in the blanks naturally**; don’t memorize word-for-word
* Say **assumptions first**, then **tradeoffs**, then **chosen solution**
* Be ready to **adjust assumptions if interviewer corrects you**
