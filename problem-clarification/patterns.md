
Common Type / Patterns
| **Type / Pattern**         | **What It Does**             | **Typical Use Cases**               |
| -------------------------- | ---------------------------- | ----------------------------------- |
| Linear Scan / Iteration    | One pass, track state        | Max, min, sum, count                |
| Two Pointers               | Coordinated pointer movement | Sorted arrays, pairs, palindromes   |
| Sliding Window             | Moving subarray              | Longest/shortest subarray, averages |
| Prefix Sum                 | Cumulative preprocessing     | Range sums                          |
| Hash Map / Frequency Count | Count or complement lookup   | Two-sum, anagrams                   |
| Set-based Lookup           | Track seen values            | Duplicates, cycles                  |
| Stack                      | LIFO processing              | Expression eval, DFS                |
| Queue                      | FIFO processing              | BFS, simulations                    |
| Binary Search              | Halve search space           | Sorted arrays, monotonic answers    |
| Depth-First Search (DFS)   | Deep traversal               | Trees, graphs, backtracking         |
| Breadth-First Search (BFS) | Level traversal              | Shortest path, grids                |
| Heap / Priority Queue      | Fast min/max access          | Top-k, scheduling                   |
| Greedy Selection           | Local optimal choice         | Interval scheduling                 |
| Interval Merge             | Combine overlaps             | Calendar, ranges                    |
| Grid DFS / BFS             | Traverse 2D grids            | Islands, flood fill                 |

| Knapsack DP                   | Constraint optimization | Weight/value          |

| Topological Sort              | Dependency ordering     | Scheduling            |
| Union-Find (Disjoint Set)     | Component tracking      | Cycle detection, MST  |
| Shortest Path Algorithms      | Weighted paths          | Dijkstra              |
| Monotonic Stack               | Ordered stack invariant | Next greater element  |
| Monotonic Queue               | Ordered queue invariant | Sliding window max    |
| Backtracking                  | Exhaustive search       | Permutations          |
| Recursion with Constraints    | DFS with rules          | Sudoku                |
| 2D Prefix Sum                 | Fast submatrix queries  | Rectangle sums        |
| Two Heaps                     | Median maintenance      | Streaming median      |
| Reservoir Sampling            | Random stream selection | Large datasets        |
