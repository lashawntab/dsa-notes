Common Domains
| **Domain**                  | **Typical Data Structures**                   | **One-Line Usage / Pattern**                           |
| --------------------------- | --------------------------------------------- | ------------------------------------------------------ |
| **Arrays / Lists**          | Array, List, Dynamic Array                    | Sequence storage, iterating/scanning, random access    |
| **Strings / Text**          | String, Char Array, Trie                      | Substring search, pattern matching, prefix problems    |
| **Numbers / Math**          | Integer, Float, Bitmask                       | Counting, arithmetic, subset or combinatorial problems |
| **Stacks / Undo / History** | Stack                                         | Evaluate expressions, DFS simulation, undo operations  |
| **Queues / Scheduling**     | Queue, Deque, Priority Queue                  | BFS, task scheduling, sliding window                   |
| **Hashing / Fast Lookup**   | HashMap, HashTable, Dictionary                | Frequency counting, caching, key-value retrieval       |
| **Sets / Uniqueness**       | Set, HashSet, BitSet                          | Track unique elements, membership checks               |
| **Trees / Hierarchies**     | BST, Heap, Trie                               | Search, priority access, hierarchical or prefix data   |
| **Graphs / Networks**       | Adjacency List/Matrix, Edge List              | BFS/DFS, shortest path, connectivity                   |
| **Matrix / 2D Grid**        | 2D Array, Sparse Matrix, Graph Representation | DFS/BFS, DP on grids, flood fill, pathfinding          |

Advanced Domains
| **Domain**                           | **Typical Data Structures**                    | **One-Line Usage / Pattern**                           |
| ------------------------------------ | ---------------------------------------------- | ------------------------------------------------------ |
| **Intervals / Ranges**               | Interval Tree, Segment Tree, Fenwick Tree      | Range queries, overlaps, bookings                      |
| **Multidimensional / Spatial**       | KD-Tree, QuadTree, Segment Tree                | Nearest neighbor, collision detection, spatial queries |
| **Combinatorial / Subsets**          | Bitmask, Array of Counters, Recursion          | DP on subsets, backtracking, subset generation         |
| **Graphs with Weights / Priorities** | Min/Max Heap, Fibonacci Heap                   | Weighted shortest path, event prioritization           |
| **Time / Event Systems**             | Heap, Queue                                    | Event simulation, scheduling with deadlines            |
| **Caching / Recent Access**          | LRU Cache (LinkedHashMap + Doubly Linked List) | Memoization, caching frequently used data              |
| **Graph Connectivity / Components**  | Union-Find (Disjoint Set), DFS, BFS            | Connected components, Kruskal MST                      |
| **String Matching / Patterns**       | KMP Table, Suffix Tree, Trie                   | Efficient pattern searches, DNA sequences              |
| **Dynamic / Modifiable Sequences**   | Linked List, Skip List, Doubly Linked List     | Fast insertions/deletions, ordered collections         |
| **Priority-based Operations**        | Priority Queue, Heap                           | Task prioritization, median finding                    |
